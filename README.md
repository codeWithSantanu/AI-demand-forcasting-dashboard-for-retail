# 🚀 AI-Powered Retail Demand Forecasting Dashboard

📊 A full-stack AI-driven application designed to help retailers predict product demand, optimize inventory levels, and make data-driven business decisions.

This system leverages **Machine Learning models, time-series forecasting (LSTM), and interactive dashboards** to provide accurate demand predictions and actionable insights.

⚡ Built using modern technologies like **Next.js (Frontend), FastAPI (Backend), and Python-based ML models**, with support for scalable deployment using Docker and cloud platforms.

👉 Helps reduce overstocking, prevent stockouts, and improve overall retail efficiency.
